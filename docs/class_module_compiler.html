static

Create a ModuleCompiler.

This is a static factory because std::enable_shared_from_this<ModuleCompiler> requires a shared_ptr to already exist before being used. This method guarantees that at least one such pointer exists. It also handles creation of the OperatorCodegen, since that also requires a shared_ptr of this. Calls addMainFunction.
