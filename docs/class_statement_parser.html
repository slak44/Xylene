Detailed Description

Provides the statement method for parsing an a statement.

Depends on ExpressionParser for parsing expressions. Depends on IfStatementParser for parsing if statements. Depends on BlockParser for parsing blocks. Depends on DeclarationParser for parsing declarations. Depends on FunctionParser for parsing function definitions. Depends on TypeParser for parsing type definitions.
