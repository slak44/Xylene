Detailed Description

Provides the block method for parsing a block.

Depends on StatementParser for parsing individual statements.

Member Function Documentation

Node< BlockNode >::Link BlockParser::block ( BlockType  type)

Parse a block, depending on its type.

Normal code blocks are enclosed in K_DO/K_END. Root blocks aren't expected to be enclosed, and instead they end at the file end. If blocks can also be ended with K_ELSE besides K_END.
